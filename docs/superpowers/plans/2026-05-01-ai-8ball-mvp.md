# AI 8-Ball MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a frontend-only AI-style 8-ball MVP with text input, browser voice input, contextual mock answers, cursor/touch shaking, mobile motion shaking, and a polished minimal interface.

**Architecture:** Use a Vite React app with focused modules for the mock decision engine, shake energy math, speech recognition wrapper, and UI components. Keep real AI out of the MVP by preserving a future-compatible `{ options: string[] }` contract.

**Tech Stack:** Vite, React, TypeScript, Vitest, Testing Library, CSS.

---
